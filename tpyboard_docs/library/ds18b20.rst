:mod:`DS18B20` --- 开发板关联功能函数
=============================================

.. module:: DS18B20
   :synopsis: 开发板关联功能函数

`` DS3221`` 模块的主要功能与函数
功能相关函数
----------------------

.. function:: read_temp()

   读取温度值，返回值为浮点型

DS18B20与开发板接线对应引脚：
----------------------

		+------------+---------+
		| TPYBoard   | DS18B20  |
		+============+=========+
		| GND        | GND     |
		+------------+---------+
		| 3V3        | VCC     |
		+------------+---------+
		| Y10        | D0      |
		+------------+---------+

程序示例：
----------

.. code-block:: python

  # main.py -- put your code here!
  from pyb import Pin
  from ds18x20 import DS18X20
  wd = DS18X20(Pin('Y10'))

  while True:
    print(当前温度:',wd.read_temp())
    pyb.delay(1000)
