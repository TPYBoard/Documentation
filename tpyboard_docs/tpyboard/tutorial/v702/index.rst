**注意：**
 - **v702开发板增加了板载的SHT20温湿度传感使用开发板上的I2C 1(SCL = X9, SDA = X10)。**
 - v702开发板上的LCD5110的接口连接的是SPI 1。
    - DIN   = X8(MOSI)
    - CLK   = X6(SCK)
    - RST   = X20
    - CE    = X19
    - DC    = X18
    - LIGHT = X17
 - **v702开发板上的GU620模块使用了UART 4(TX = X1, RX = X2)。**
 - **v702开发板上的光敏电阻连接的是引脚Y12。**
 - **v702开发板上的蜂鸣器模块连接的是引脚Y3。**
