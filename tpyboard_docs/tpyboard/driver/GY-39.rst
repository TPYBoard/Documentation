:mod:`GY-39/MAX44009+BME280` --- 气象监测模块关联功能函数
=============================================

.. module:: GY-39
   :synopsis: 气象监测模块关联功能函数

``gy39`` 模块的主要功能与函数

功能相关函数
----------------------

.. only:: port_tpyboard

    .. class:: gy39.GY39(id)
    
    创建GY39对象。
    
        - ``id`` 硬件连接的UART编号
    
    .. method:: GY39.read()

       读取当前环境函数，返回值为光照强度,温度,气压,湿度,海拔高度

GY-39与开发板接线对应引脚：
------------------------------------

		+------------+---------+
		| TPYBoard   | GY-39   |
		+============+=========+
		| X9         | DR      |
		+------------+---------+
		| X10        | CT      |
		+------------+---------+
		| 3V3        | VCC     |
		+------------+---------+
		| GND        | GND     |
		+------------+---------+

程序示例：
------------

.. code-block:: python

  import pyb
  from gy39 import GY39
  import time
  gy_39=GY39(1)
  
  while True:
  	l,t,p,h,H=gy_39.read()
  	print('光照强度:%s lux'%l)
  	print('当前温度:%s ℃'%t)
  	print('当前湿度:%s ％'%h)
  	print('当前气压:%s pa'%p)
  	print('海拔高度:%s m'%H)
  	print('')
  	time.sleep_ms(200)
