``GY-39`` 模块的主要功能与函数

功能相关函数
----------------------

.. function:: read(self)

   读取当前环境函数，返回值为光照强度,温度,气压,湿度,海拔高度

GY-39与开发板接线对应引脚：
------------------------------------

		| X10        | CT      |

		+------------+---------+

程序示例：
------------

.. code-block:: python

  import pyb
  import GY39
  import time
  gy39=GY39.GY39(1)
  
  while True:
  	l,t,p,h,H=gy39.read()
  	print('光照强度:%s lux'%l)
  	print('当前温度:%s ℃'%t)
  	print('当前湿度:%s ％'%h)
  	print('当前气压:%s pa'%p)
  	print('海拔高度:%s m'%H)
  	print('')
  	time.sleep_ms(200)
